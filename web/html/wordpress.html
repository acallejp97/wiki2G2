Para empezar descargaremos wordpress en el servidor. Para ello metemos el siguiente comando: 

	wget https://wordpress.org/versionactual.tar.gz /var/www/html

De esta manera descargaremos el zip en la carpeta html que es en la cual trabajaremos ahora.

Una vez nos hayamos movido de esta carpeta lo que haremos será descomprimir el archivo mediante el comando 

	tar -xzvf versionactual.tar.gz

Ahora crearemos una base de datos para nuestro wordpress la cual almacenará todos los datos de nuestra pagina web, para ello introducimos los siguientes comandos:

	mysql -u root 

	CREATE DATABASE wordpress;
	CREATE USER 'wordpress'@'localhost' IDENTIFIED BY 'grupo2';
	GRANT ALL PRIVILEGES ON wordpress.* TO 'wordpress'@'localhost IDENTIFIED BY 'grupo2;
	FLUSH PRIVILEGES;
	EXIT;

Una vez hayamos creado la base de datos, lo que tendremos que hacer será preparar el entorno para poder instalar la pagia web. Para ello nos moveremos a la carpeta /var/www/html/wordpress y renombraremos el archivo wp-config-sample.php a wp-config.php. Luego lo editaremos y rellenaremos los campos con los datos correspondientes a la base de datos.

Ya terminado con esto, iremos a la IP del servidor y empezaremos con la instalacion de wordpress
